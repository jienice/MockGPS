<h3>模拟iPhone定位</h3>

1、新建工程

2、创建`.gpx`文件

3、查询需要模拟位置的经纬度

4、编辑文件

```
    <wpt lat="31.1938100000" lon="121.4362380000">
        <name>徐家汇地铁站</name>
        <time>2017-11-09T14:55:37Z</time>
    </wpt>
```

5、`edit scheme -->Options-->Default Location` 选择创建的`.gpx`文件名称
